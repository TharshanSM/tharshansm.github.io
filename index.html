05 — Projects

What I build in my spare time

IFS Debug Console

In development · Chrome Extension

A Chrome extension that adds a debug side panel to IFS Cloud, built in my spare time with AI-assisted development. It signs in through an IAM client using OAuth (Authorization Code + PKCE) and works entirely over IFS REST services — an API console with metadata-driven suggestions, a field inspector that shows the real database column and view behind any field on the page, a read-only SQL editor, a network trace that surfaces the PL/SQL error stack behind failed OData calls, and a projection tracer that maps every sub-page of a record.

Chrome Extension (MV3) JavaScript IFS Cloud REST / OData OAuth 2.0 (PKCE) IAM (Keycloak)
